#_____| Gears |________________________________________________
кнопка create join создаёт ограничитель в цель надо вставить главный объект(от которого идёт вращение)
а delete join удаляет ограничитель. (-)
Объект с ограничителем не возможно вращать.

Кнопка create join (side) то же, что и create join, но для горизонтали. (|)
кнопка join (shaft) она удаляет реверсию вращение.

#_____| другое |________________________________________________
кнопка Block блокирует перемещение и поворот. Unblock разблокирует.
